响应式设计的步骤： 1.设置meta标签：大多数的浏览器将html页面放大为宽的视图（viewport）以符合屏幕分辨率， 上面的标签告诉浏览器 ，使用设备的宽度作为视图的宽度并且禁止初始的缩放 2.通过媒介查询来设置样式@media 响应式设计的核心就是@media @media screen and （max-width：980px）{ #header{} #content{} #footer{} } 通过尺寸来设置样式 3.宽度需要使用百分比 例如：#header{ width:100%} #content{width:50%} 处理图片缩放的方法 可以使用 百分比 但是 不友好，会放大或者缩小图片 可以给图片最大的宽为百分比
